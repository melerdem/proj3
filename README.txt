
README - Build Instructions (CLion + ImGui + SFML)
--------------------------------------------------

1. Clone or download ImGui-SFML:
   git clone https://github.com/eliasdaler/imgui-sfml.git vendor/imgui-sfml

2. Open this folder in CLion

3. Let CMake configure and build the project

4. Run the app

Dependencies:
- SFML installed (https://www.sfml-dev.org/download.php)
- Git (for cloning imgui-sfml)

Optional (on Linux):
sudo apt install libsfml-dev

Optional (on Mac via Homebrew):
brew install sfml
